Secure Mosquitto Broker Setup (TLS)

Platform: Raspberry Pi OS Bookworm

Certs Location: /etc/mosquitto/certs/

1. Create the password file

sudo mosquitto_passwd -c /etc/mosquitto/passwd admin

When prompted, use your secure password (e.g., admin1234S).

2. Remove conflicting configs

sudo rm -f /etc/mosquitto/conf.d/password.conf

3. Create the secure TLS config

sudo nano /etc/mosquitto/conf.d/secure-wss.conf

Paste this:

listener 8884

cafile /etc/mosquitto/certs/ca.crt
certfile /etc/mosquitto/certs/server.crt
keyfile /etc/mosquitto/certs/server.key

require_certificate false

allow_anonymous false
password_file /etc/mosquitto/passwd

4. Restart Mosquitto

sudo systemctl restart mosquitto

5. Check that Mosquitto is listening

sudo lsof -i :8884

6. Test MQTT TLS connection

Subscriber:

sudo mosquitto_sub -h localhost -p 8884 \
  --cafile /etc/mosquitto/certs/ca.crt \
  -t test -u admin -P admin1234S --insecure

Publisher:

sudo mosquitto_pub -h localhost -p 8884 \
  --cafile /etc/mosquitto/certs/ca.crt \
  -t test -m "Hello TLS MQTT!" -u admin -P admin1234S --insecure
